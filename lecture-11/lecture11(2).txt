Logical Operators
====================


&& Logical and

cond1 true && cond2 true  // result is true

cond1 true && cond2 false  // result is false

cond1 false && cond2 true  // result is false

cond1 false && cond2 false // result is false

Example
 a=10, b=5, c=15;
SOP(a>b && b>c) // false 

|| Logical or

cond1 true || cond2 false  // result is true

cond1 false || cond2 true  // result is true

cond1 falsE || cond2 false  // result is false


cond1 true || cond2 true  // result is true


Example
 a=10, b=5, c=15;
SOP(a>b || b>c) // true
