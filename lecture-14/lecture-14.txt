How do we create objects in java?

objects are used to represent real world items

class Student
{
    int roll;
    char grade;
    double per;
    int marks;

}

(in java these are called attributes,properties,instance members)

Syntaxt of creating object
==========================

<class_name> <reference> = new <class_name> ();

Example
==========
Student s=new Student();
(objects live in a special place in RAM called heap area)(rest code is in stack area)(for now)
